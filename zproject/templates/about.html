Project details

This Flask app is a SIP calculator for estimating long-term investment growth. It computes total invested amount, expected wealth gain, and projected portfolio value based on monthly deposits, annual return rate, investment duration, and yearly increment.

The app uses Bootstrap for responsive styling and inherits a common navbar from base.html so the navigation stays consistent across pages.
